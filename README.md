# courseselectingsystem

# 学生选课系统

## 项目简介：

在线考试系统旨在为学生提供一个在线考试的平台,在这里,学生不仅可以网上在线考试,而且可以浏览题库中心下的各个课程的题库,也可以在讨论区下发表自己的想法,当然也在帖子下留言评论。如果你是spring boot初学者，那么此项目非常适合你。如果喜欢，请随手给个star，谢谢。
